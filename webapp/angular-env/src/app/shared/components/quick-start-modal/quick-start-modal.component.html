  • Используйте основную страницу для ознакомления с системой.
  • Отслеживайте наполнение системы первичными данными и отчетами об анализе на страницах Отчеты и Интервью.
  • Проверяйте уведомления, чтобы получать информацию о системых событиях и статусы Ваших задач.
